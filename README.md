Phishing Awareness Analyzer 🎣
Description

A phishing detection and awareness application that analyzes user-provided messages, emails, SMS content, or URLs for common phishing indicators.

Features:
Detection of suspicious keywords
Detection of urgency language
Detection of requests for sensitive information
URL identification using Regular Expressions (Regex)
Suspicious domain detection
Risk scoring system

Risk classification:
Low Risk
Medium Risk
High Risk
Detailed explanations of detected threats

Technologies Used
Python
Tkinter
Regular Expressions (re)

Cybersecurity Concepts:
Social Engineering
Threat Detection
Phishing Analysis
Security Awareness
Risk Assessment
